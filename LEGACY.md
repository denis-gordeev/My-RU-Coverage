# Legacy-корпус (Тайвань)

Репозиторий хранит исторический тайваньский корпус в `Pilot_Reports_LEGACY/` для обратной совместимости. Этот файл описывает_legacy-поток, который **не является основным путём развития**.

## Содержимое

- `Pilot_Reports_LEGACY/` — карточки тайваньских эмитентов (тикеры формата `NNNN_名稱.md`).
- `task.md` — разбивка на batch-задачи для тайваньского корпуса.
- `scripts/audit_batch.py` — аудит качества по batch-задачам из `task.md`.

## Скрипты, работающие с legacy-корпусом

```bash
python scripts/audit_batch.py 101 -v
python scripts/audit_batch.py --all -v
```

Эти скрипты читают номера тикеров из `task.md` и ищут соответствующие файлы в `Pilot_Reports/` и `Pilot_Reports_LEGACY/`.

## Правила для legacy-файлов

- Не ломайте совместимость ради косметики.
- Финансовый раздел не редактируется вручную.
- Единицы для `.TW/.TWO`: `млн тайв. долл.`
- Язык описаний в legacy-файлах может быть китайским.

## Миграция

При необходимости мигрировать legacy-карточку в российский контур:

1. Создайте новый файл через `python scripts/add_ticker.py`
2. Переведите описания на русский
3. Обновите wikilinks и цепочку поставок под российские реалии
