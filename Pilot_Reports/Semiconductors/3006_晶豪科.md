# 3006 - 晶豪科

## 業務簡介
晶豪科 (ESMT) 是台灣老牌的 **利基型記憶體 (Niche Memory) IC設計公司**。
主要產品包括 SDRAM, DDR1/2/3, Flash Memory 等，應用於[[消費性電子]]、[[網通]]設備、[[物聯網]]裝置等「非PC主流」市場。
與一般標準型[[記憶體]] (如三星、美光主導這PC/Server市場) 不同，利基型[[記憶體]]強調客製化與供應穩定。
晶豪科近年亦拓展 **類比/混合訊號 IC** (如音頻放大器、電源管理IC)，採「Memory + Analog」雙軌策略。

## 供應鏈位置
*   **IC設計 (本體)**: **晶豪科 (3006)** - 負責晶片設計與銷售。
*   **上游**: 晶圓代工廠 (力積電、武漢新芯等) 負責製造。
*   **下游**: 模組廠、[[消費性電子]]系統廠、[[網通]]設備廠。

## 主要客戶及供應商
### 主要客戶
*   **應用端**:機上盒 (Set-top box)、WiFi 路由器、智慧音箱、硬碟 (HDD Buffer) 廠商。

### 主要供應商
*   **晶圓代工**: 力積電 (長期合作夥伴)。
