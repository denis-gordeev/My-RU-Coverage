# 3659 - 百辰

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 1,873 百萬台幣
**企業價值:** 1,867 百萬台幣

百辰光電 (Bison) 為專業的 **鏡頭模組 (Camera Module)** 製造商，主要應用於筆記型電腦 (NB) 與電腦周邊產品。
公司由昆山百創光電等子公司組成，深耕光學儀器與資訊設備。
業務重點與發展：
1.  **電腦鏡頭模組**: 營收高佔比 (約 95% 以上) 來自於電腦相機模組，是各大品牌 NB 供應鏈的重要成員。
2.  **多元化影像應用**: 產品線延伸至工業用手持裝置、智慧醫療產品及智慧家居安全監視系統。
3.  **車用影像佈局**: 積極研發生產車用鏡頭模組及駕駛輔助警告系統，尋求在非 3C 領域的長期成長動力。
4.  **精密元件製造**: 具備從模具到組裝的垂直整合能力，提供輕量化、窄邊框設計所需之微型鏡頭模組。

## 供應鏈位置
*   **上游**: 感測器 (Sensor), 光學鏡片, 音圈馬達 (VCM), [[PCB]] 基板。
*   **中游**: **百辰 (3659)** - 鏡頭模組開發、封裝與組裝。
*   **下游**:
    *   **筆電代工廠**: [[廣達]], [[仁寶]], [[緯創]].
    *   **工業與車用**: [[工業電腦]]大廠, 車載系統整合商。
    *   **智慧居家品牌商**。

## 主要客戶及供應商
### 主要客戶
*   全球一線筆記型電腦品牌業者。
*   工業手持設備及醫療影像器材品牌商。
