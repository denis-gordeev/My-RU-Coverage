# 3652 - 精聯

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 2,159 百萬台幣
**企業價值:** 1,288 百萬台幣

精[[聯電]]子 (unitech) 為全球領先的 **自動辨識及資料收集 (AIDC)** 解決方案供應商。
原屬精技電腦 (2414) 之自動資料收集事業部，於 2008 年獨立。以自有品牌 **「unitech」** 行銷全球，擁有逾 45 年的產業經驗。
核心業務與亮點：
1.  **強固型行動電腦**: 生產具備高防護等級 (IP65/IP67) 的掌上型終端機 (PDA) 與工業用平板，支援 Android 與 Windows 系統。
2.  **條碼掃描與 RFID**: 提供全系列條碼掃描器（手持、穿戴、口袋型）及 RFID 辨識解決方案，廣泛應用於資產管理與物流追蹤。
3.  **MoboLink 平台**: 自主開發行動設備管理 (MDM) 軟體，協助企業管理全球據點的行動終端，監測設備狀態並遠端更新。
4.  **智慧物流與倉儲**: 提供從硬體到軟體的一站式 AIDC 解決方案，服務零售、醫療、配送與製造等產業。

## 供應鏈位置
*   **上游**: 處理器、顯示面板、掃描引擎裝置、塑橡膠與金屬機構件、板卡元件。
*   **中游**: **精聯 (3652)** - AIDC 產品研發設計、系統整合與自有品牌經營。
*   **下游**:
    *   **零售與電商**: [[Walmart]], [[IKEA]], [[蝦皮]].
    *   **物流與運輸**: 跨國物流業者, 各國國家郵政系統。
    *   **醫療與公用**: 大型醫療體系, 政府機構, 軍警單位。

## 主要客戶及供應商
### 主要客戶
*   Walmart, IKEA 等全球零售龍頭。
*   澳洲連鎖超市與國際大型快遞物流公司。
*   國內知名[[半導體]]大廠及昇恆昌。
