# 3022 - 威強電

## 業務簡介
威強電 (IEI Integration) 是 **工業電腦 (IPC)** 大廠，主要股東為威聯通 (QNAP)。
產品涵蓋工業[[自動化]]、[[網通]]設備 (Networking) 與 **智慧醫療 (Healthcare)**。
醫療領域深耕多年，提供醫療級 Panel PC、護理推車電腦、內視鏡影像系統等，獲 ISO13485 認證。
近年積極佈局 **Edge AI** 邊緣運算，與 [[Intel]] 合作緊密，推出 [[AI]] 推論系統。
[[網通]]產品 (Puzzle Series) 亦是重要營收來源，應用於 SD-WAN、防火牆與 uCPE。

## 供應鏈位置
*   **工業電腦品牌/代工廠 (本體)**: **威強電 (3022)** - 系統整合與設計。
*   **上游**: CPU ([[Intel]], [[AMD]])、面板、[[DRAM]]、[[PCB]]、機殼。
*   **下游**: 
    1.  **醫療**: 國際醫療設備大廠 ([[ODM]])、醫院。
    2.  **網通**: 資安軟體商、電信營運商。
    3.  **工控**: 智慧工廠系統整合商。

## 主要客戶及供應商
### 主要客戶
*   **醫療 ODM**: 歐美醫療設備與耗材大廠。
*   **網通**: 國際[[網通]]與資安客戶。

### 主要供應商
*   **處理器**: [[Intel]]。
