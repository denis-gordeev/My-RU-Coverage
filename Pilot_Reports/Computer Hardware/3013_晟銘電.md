# 3013 - 晟銘電

## 業務簡介
晟銘電 (Uneec) 是 **AI 伺服器機殼與液冷散熱** 的重要概念股，與[[廣達]] (Quanta) 關係緊密。
早期做 PC 機殼，近年成功轉型[[伺服器]]機殼與 L6 (Barebone) 組裝。
在 **Meta (Facebook)** 的 [[AI]] [[伺服器]]供應鏈中扮演關鍵角色，負責供應 **Sidecar 水冷機櫃** 與機箱。
擁有自主研發的 L10/L11 液冷機櫃組裝能力，受惠於 GB200 等高階 [[AI]] [[伺服器]]需求爆發。

## 供應鏈位置
*   **機構件/散熱供應商 (本體)**: **晟銘電 (3013)** - 機殼、[[水冷]] Sidecar、L6 組裝。
*   **上游**: 金屬板材、[[水冷]]零組件 (CDU, Manifold - 部分自製或外購)。
*   **下游**: [[ODM]] 廠 ([[廣達]]、[[緯創]]、英業達) -> [[CSP]] ([[Meta]], ByteDance)。

## 主要客戶及供應商
### 主要客戶
*   **ODM**: [[2382_廣達|廣達]] (主力客戶)、[[3231_緯創|緯創]]。
*   **終端**: [[Meta]] (Facebook) - Sidecar 專案, ByteDance。

### 主要供應商
*   **原料**: 鋼材、鋁材。
