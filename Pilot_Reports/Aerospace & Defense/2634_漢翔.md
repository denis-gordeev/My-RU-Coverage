# 2634 - 漢翔

## 業務簡介
漢翔航空工業 (AIDC) 前身為空軍航發中心，是台灣航太產業龍頭與國防自主核心。公司業務涵蓋軍用飛機 (如 F-16 構改、[[勇鷹號]]高教機生產)、民用飛機機體結構與發動機零組件製造，以及機電能源專案。
漢翔是美國 [[Lockheed Martin]] 在亞洲唯一授權成立的 **F-16 維修中心**，並承擔政府「國機國造」政策重任，同時也是波音 (Boeing) 與空中巴士 (Airbus) 的重要 Tier 1/Tier 2 供應商。

## 供應鏈位置
*   **上游**: 航太鋁合金/複材供應商、特殊製程原物料、次系統供應商。
*   **中游**: **漢翔 (2634)** - 飛機結構組裝、發動機組件製造、全機系統整合、MRO (維修) 服務。
*   **下游**:
    *   **國防**: [[中華民國空軍]] (ROCAF) - F-16V 升級維修、勇鷹號高教機交付。
    *   **民航**: 國際航太巨頭 ([[Boeing]], [[Airbus]], [[Bombardier]])、發動機大廠 ([[GE Aviation]], [[Rolls-Royce]]).

## 主要客戶及供應商
### 主要客戶
*   **國防部/空軍**: 最大單一客戶，涵蓋鳳展專案 (F-16V構改)、勇鷹高教機量產、F-16 維修中心業務。
*   **國際民航客戶**:
    *   **Boeing**: B737, B747, B777 機體結構或艙門。
    *   **Airbus**: A320 系列機腹整流罩等複材組件。
    *   **GE / Rolls-Royce / Honeywell**: 發動機機匣與葉片零組件。

### 主要供應商與供應鏈夥伴
*   **勇鷹號供應鏈**: 漢翔整合國內 107 家廠商建立「國防台灣隊」，包括 [[千附精密]] (結構件)、[[駐龍]] (機身結構)、[[晟田]] (起落架零件)、[[拓凱]] (複材) 等。
*   **原物料**: 向國外認證供應商採購航太級鋁合金、鈦合金與碳纖維。
