# 5240 - 建騰

## 業務簡介
**板塊:** Industrials
**產業:** Security & Protection Services
**市值:** 501 百萬台幣
**企業價值:** 361 百萬台幣

建騰創達 (ACTi) 專注於 **IP 網路監控系統** 之研發，提供從攝影機到管理平台的端到端解決方案。
核心業務：
1.  **網路攝影機 (IP Cameras)**: 涵蓋槍型、半球型、快速球型等全系列產品。
2.  **影像管理系統 (VMS/NVR)**: 錄影[[伺服器]]軟硬體。
3.  **AI 影像分析**: 導入邊緣運算 (Edge [[AI]])，提供人流計數、車牌辨識、異常偵測等功能。
4.  **雲端服務**: 與 [[Microsoft]] Azure 合作推出雲端監控方案。
產品強調符合 **NDAA (國防授權法案)** 規範，主攻非中國制裁市場。

## 供應鏈位置
*   **中游 (本體)**: **建騰 (5240)** - 系統設計, 軟體開發, 硬體整合.
*   **上游**:
    *   **關鍵晶片**: SoC (符合 NDAA 規範之晶片), 鏡頭模組, 感測器.
*   **下游**:
    *   **系統整合商 (SI)**: 負責終端安裝與維護.
    *   **終端場域**: 零售 (智慧商店), 工廠, 銀行, 政府機關.

## 主要客戶及供應商
### 主要客戶
*   **海外**: 美國及全球各地之安防系統整合商/代理商.
*   **專案**: 零售連鎖店, 企業園區.

### 主要供應商
*   **晶片**: 非紅色供應鏈之 IC 設計廠.
