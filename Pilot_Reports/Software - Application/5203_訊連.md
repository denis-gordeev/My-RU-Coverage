# 5203 - 訊連

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 6,034 百萬台幣
**企業價值:** 3,774 百萬台幣

訊連科技（5203）是全球知名的多媒體影音軟體公司，亦是台灣軟體產業的領導者。主要產品包括「威力導演」（PowerDirector）、「相片大師」（PhotoDirector）等創意編輯軟體，以及FaceMe® [[AI]]臉部辨識技術。公司近年成功從傳統買斷制轉型為訂閱制（SaaS），並積極投入生成式[[AI]]（GenAI）技術研發，應用於數位行銷及內容創作。業務範圍涵蓋B2C（消費者軟體）及B2B（企業解決方案、FaceMe授權）。

## 供應鏈位置
作為軟體開發商，訊連在供應鏈中屬於終端應用提供者。
- **合作夥伴:** 
    - **硬體OEM:** 與[[HP]], [[Dell]], Acer, [[Lenovo]], Asus等PC大廠長期合作，預載軟體於電腦中。
    - **技術夥伴:** [[Intel]], [[NVIDIA]], [[Microsoft]]（[[AI]]技術與硬體優化合作）。
    - **雲端服務:** 使用AWS等雲端基礎設施。

## 主要客戶及供應商
- **主要客戶:**
    - **B2C:** 全球個人創作者、Youtuber、家庭用戶（透過訂閱制）。
    - **B2B (OEM):** 全球主要PC品牌商（[[HP]], [[Dell]], Acer等）。
    - **B2B (FaceMe):** 智慧零售、安控、金融機構（如元大銀行）、智慧辦公室解決方案採用者。
- **主要供應商:** [[雲端服務]]供應商（AWS）、應用程式商店平台（App Store, [[Google]] Play）。
