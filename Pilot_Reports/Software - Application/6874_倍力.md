# 6874_倍力

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 1,451 百萬台幣
**企業價值:** 1,652 百萬台幣

倍力資訊股份有限公司 (6874) 是一家專注於企業級軟體代理與自主研發ESG數位轉型解決方案的資訊服務旗艦廠。倍力資訊過去以代理國際頂級資料庫與網通軟體(如Oracle、SAP)在業界奠定穩固地位，掌握極佳的客戶覆蓋率。近年來，公司洞悉金融法規趨勢，成功轉型並自主研發「智慧財務解決方案」(協助上市櫃公司快速產出IFRS合併報表)以及「碳管家(CarbonKeeper)」(企業碳盤查與減碳模擬系統)。倍力精準擊中了台灣數千家上市櫃公司必須強制接軌國際永續準則與財報合規的剛性痛點，創造極高市佔率。

## 供應鏈位置
*   **上游**: 國際頂尖微軟、資料庫、網通及企業資源規劃(ERP)等大型商用軟體品牌原廠。
*   **中游**: **倍力資訊** (國際商用基礎軟體代理經銷及自主研發ESG碳管家與IFRS集團合併報表SaaS軟體服務)。
*   **下游**: 受到金管會強制規範撰寫永續報告書及合併報表的台灣所有上市櫃公司、金融機構及大型製造業。

## 主要客戶及供應商
### 主要客戶
*   以金融業、電信業及大型高科技製造業為主(如聯強、新光三越等)，累計服務超過400家上市櫃企業。

### 主要供應商
*   全球各大資訊軟體原廠，包含[[Oracle]](甲骨文)、[[SAP]]、Cisco、IBM等世界級軟硬體巨擘。
