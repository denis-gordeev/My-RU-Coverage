# 5202 - 力新

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 1,292 百萬台幣
**企業價值:** 789 百萬台幣

力新國際 (NewSoft) 隸屬於 **台灣鋼鐵集團 (TSG)**，為台灣知名軟體公司。
早年以 **OCR (光學字元辨識)** 軟體「丹青 (Presto! PageManager)」與「非常好色」聞名。
目前業務轉型為企業數位轉型服務商：
1.  **AI OCR/IDR**: 企業文件辨識與[[自動化]]輸入系統 (智慧單據辨識)。
2.  **企業解決方案**: BPM (流程管理), 雲端辦公室 (Pro OA), 碳盤查系統。
3.  **系統整合**: 配合母集團台鋼資源，進行相關資訊系統整合。
4.  **AIoT**: 智慧聯網服務。

## 供應鏈位置
*   **中游 (本體)**: **力新 (5202)** - 軟體產品開發 & 企業解決方案.
*   **上游**: 雲端平台供應商 (AWS/Azure), 硬體合作夥伴 (掃描器/事務機廠).
*   **下游**:
    *   **企業客戶**: 金融業, 製造業, 政府機關, 學校.
    *   **集團內部**: 台鋼集團各子公司之數位轉型需求.

## 主要客戶及供應商
### 主要客戶
*   **金融**: [[第一金]], [[富邦金]].
*   **科技/製造**: [[Gogoro]], [[亞尼克]], 台鋼集團成員.

### 主要供應商
*   **雲端**: [[Microsoft]] Azure, AWS.
*   **硬體**: 掃描器廠商 (搭售軟體).
