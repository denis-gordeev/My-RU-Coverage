# 6902_GOGOLOOK

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 2,790 百萬台幣
**企業價值:** 2,741 百萬台幣

走著瞧股份有限公司 (6902，Gogolook) 是一家名揚國際以「信任科技(TrustTech)」為核心的AI軟體服務(SaaS)獨角獸。Gogolook從台灣國民App「Whoscall」起家，透過用戶回報與AI大數據分析，累積了全球龐大的電話識別與防詐騙惡意資料庫。隨後，公司成功佈局企業端(如收購ScamAdviser提供全球企業假網址與商譽保護風險偵測)、個人端防詐達人(美玉姨查核系統)以及以大數據信用評級切入的金融科技媒合服務(袋鼠金融 Roo.Cash)。公司深受Line(NAVER)的資金挹注與技術肯定，亦是多國警政安保單位的御用合作夥伴。

## 供應鏈位置
*   **上游**: 全球大型公有雲端計算與數據儲存資源供應商(如AWS及Google)、各國基礎電信營運商(提供API串接線路平台)。
*   **中游**: **走著瞧(Gogolook)** (通訊軟體Whoscall來電辨識系統研發、企業端全球防詐雲端分析ScamAdviser及金融商品媒合平台)。
*   **下游**: 飽受通訊詐騙之苦的全球廣大智慧手機消費者、需保護商譽防堵釣魚網站的跨國企業品牌、及推廣信貸的金融機構。

## 主要客戶及供應商
### 主要客戶
*   龐大的App訂閱消費者；企業端防詐客戶包含凱基證券與中信等大型金控；政府端更橫跨台灣警政署與日本、泰國警方。

### 主要供應商
*   初期最大外部控股集團與策略合作對象NAVER (Line母公司)，以及為其「防詐聯盟」提供算力基礎的多國雲端服務巨擘。
