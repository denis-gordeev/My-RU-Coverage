# 6882_甲尚

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 413 百萬台幣
**企業價值:** 222 百萬台幣

甲尚股份有限公司 (6882) 是一家在全球角色動畫與多媒體軟體工具領域享有盛譽的台灣企業。甲尚專注於降低高階動畫製作的門檻，其開發的軟體工具具備強大的人機互動與AI臉部/肢體動作捕捉分析核心技術，讓電影級別的3D角色建模與動畫生成，即使是獨立影片創作者或小型工作室也能輕易完成。在這個元宇宙與虛擬主播爆發的時代，甲尚更是從單純的軟體買賣，成功轉型為提供海量數位內容(服裝、動作庫)銷售的全球大型資產平台。

## 供應鏈位置
*   **上游**: 各大雲端運算AI大語言與圖形生成演算模型、虛擬實境動作捕捉硬體商以及程式開發與3D繪圖引擎。
*   **中游**: **甲尚** (高階角色動畫創建軟體、臉部與人體動作捕捉技術研發及全球數位內容素材庫平台營運)。
*   **下游**: 全球影視動畫製作公司、頂尖與獨立遊戲開發商、虛擬網紅經濟、廣播電視網路媒體與教育訓練機構。

## 主要客戶及供應商
### 主要客戶
*   全球無數的專業動畫師、影音創作者、遊戲開發者，甚至包含歐美一流的大型動畫電影製片廠及遊戲設計公司。

### 主要供應商
*   遍布全球為其平台提供付費3D建模與動作模組的無數獨立數位藝術家與自由接案內容開發商。
