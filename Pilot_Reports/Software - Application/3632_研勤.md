# 3632 - 研勤

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 376 百萬台幣
**企業價值:** 483 百萬台幣

研勤科技以衛星導航品牌 **「PAPAGO!」** 奠定市場地位，目前已轉型為 **AI 影像與人臉辨識** 領導廠商。
核心競爭力與轉型成果：
1.  **FACE8 臉霸**: 自主研發人臉辨識引擎，為台灣落地量最大之系統，應用於考勤、校園管理及健身房會員識別。
2.  **車載電子**: 維持「PAPAGO!」行行車紀錄器、胎壓偵測及電子後視鏡優勢，並整合 [[AI]] 影像演算法。
3.  **智慧監控**: 提供智慧工安防護雲端平台（[[AI]] Guard），偵測施工安全合規性。
4.  **多角化經營**: 透過 FarBar 廣告機佈局大型商圈，並跨足餐飲業（掌門精釀）進行品牌聯名。

## 供應鏈位置
*   **上游**: 影像感測晶片, GPS 感測器, OS 代碼庫, 門禁硬體組件.
*   **中游**: **研勤 (3632)** - [[AI]] 演算法開發 / 人臉辨識軟體平台 / 車用產品設計.
*   **下游**:
    *   **企業管理**: [[健身工廠]] (會員識別), 台中市政府 (考勤專案), 金融大型企業.
    *   **終端零售**: 全家便利商店 (智慧鎖), 汽車百貨, 電商通路.
    *   **公共工程**: 營建案場工安監修業者.

## 主要客戶及供應商
### 主要客戶
*   健身工廠 (柏文).
*   政府機關專案與校園系統.
*   大型金融機構與長照中心.
