# 6590_普鴻

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 1,785 百萬台幣
**企業價值:** 1,597 百萬台幣

普鴻資訊股份有限公司 (6590，Provision) 是台灣金融支付與資訊安全軟體系統的隱形冠軍。普鴻專注於金融業的核心支付應用、清算系統及企業級資安解決方案，在台灣銀行圈的市佔率極高(特別是票交/跨行支付系統)。有別於一般接案的軟體外包商，普鴻主打自有產品的授權與常態性維護服務(SaaS/PaaS)，創造極為穩定的高黏著度經常性收入。為響應「資安國造」，普鴻推出了ProTrust等硬體加解密與資料保護平台，並透過併購「捷智商訊」進一步稱霸金融監理科技(RegTech)市場法規報表領域。

## 供應鏈位置
*   **上游**: 國際資安硬體防護模組(HSM)原廠、全球公有雲平台(IaaS)及第三方關聯式資料庫引擎。
*   **中游**: **普鴻資訊** (金融電子支付清算平台、金融資安防護系統、監理科技報表軟體之研發、建置與維護)。
*   **下游**: 各大商業銀行、純網銀、政府金融監管機構、第三方支付業者及大型企業客戶。

## 主要客戶及供應商
### 主要客戶
*   富邦、玉山、台新等台灣幾乎所有的大型商業銀行，以及財政部、中央銀行等政府金融單位。

### 主要供應商
*   Thales(核心硬體加密模組合作夥伴)、微軟/Oracle等跨國軟體與資料庫大廠、以及專業軟體工程人才。
