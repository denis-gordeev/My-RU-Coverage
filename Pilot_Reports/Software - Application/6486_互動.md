# 6486_互動

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 4,005 百萬台幣
**企業價值:** 3,331 百萬台幣

互動國際數位股份有限公司 (6486) 隸屬於[[明基佳世達]]集團，是台灣頂尖的電信、媒體及寬頻網路系統整合商(SI)。互動國際專注於代理歐美一線硬體與軟體品牌，為客戶提供包含[[5G]]骨幹網路、微波無線傳輸、數位電視多媒體播送、地理資訊系統(GIS)及雲端資安的完整系統建置與售後服務。隨著人工智慧、[[低軌衛星]]通訊及[[5G]]企業專網的蓬勃發展，互動國際成功將業務觸角從傳統電信商(Telco)專案，延伸至高鐵軌道通訊、智慧城市防災與智慧城鄉等大型基礎設施智慧化之建置案。

## 供應鏈位置
*   **上游**: 國外各大網路設備、微波通訊設備、多媒體製播軟硬體設備及地理資訊軟體原廠。
*   **中游**: **互動國際** (電信/寬頻網路、數位媒體及GIS地理資訊系統之代理與系統整合建置)。
*   **下游**: 電信營運商、有線/無線電視台、OTT新媒體平台、政府機構及大型軌道交通營運商。

## 主要客戶及供應商
### 主要客戶
*   中華電信等電信三雄、各大有線/無線電視台、台灣高鐵、各縣市政府及學術研究機構。

### 主要供應商
*   Cisco、Ericsson、ESRI(全球GIS軟體龍頭)及AWS等國際一線[[網通]]與軟體設備大廠。
