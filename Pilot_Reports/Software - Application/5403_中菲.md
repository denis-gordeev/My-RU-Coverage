# 5403 - 中菲

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 8,678 百萬台幣
**企業價值:** 7,627 百萬台幣

中菲電腦 (Dimerco Data System) 成立於 1981 年，總部位於台灣[[台北]]。公司為專業的 **資訊系統整合服務商 (SI)**，專注於 **金融業**、**證券業** 及 **物流業** 之軟體開發與系統維護。主要業務涵蓋銀行信託系統、證券交易系統、貨運承攬管理系統等。中菲長期深耕利基市場，擁有高市佔率與穩定客戶群。

## 供應鏈位置
**角色:** 軟體系統整合商 (下游)。
**上游:** 硬體設備商 (IBM, [[HP]] 等)、軟體原廠 ([[Microsoft]], Oracle)。
**下游:** 銀行、證券公司、物流企業。

## 主要客戶及供應商
### 主要客戶
- **金融:** 公股行庫 (台灣銀行、土地銀行)、民營銀行 (台北富邦、國泰世華)。
- **證券:** 國內各大證券商 (信託部門系統)。
- **物流:** 中菲行 (Dimerco Express, 集團關係企業/主要客戶)。
