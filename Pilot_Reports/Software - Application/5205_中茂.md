# 5205 - 中茂

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 356 百萬台幣
**企業價值:** 304 百萬台幣

中茂能資 (Taiwan Green Environment / TGE) 前身為「漢平電子」轉投資之相關企業，後轉型投入 **環保綠能** 領域。
目前業務較為多元且控股化：
1.  **環境工程**: 水資源處理、廢棄物處理、生質能發電規劃。
2.  **通路經銷**: 電子零組件代理、民生消費品代理。
營運規模較小，且業務仍在調整轉型中。

## 供應鏈位置
*   **中游 (本體)**: **中茂 (5205)** - 環境工程整合 & 產品代理.
*   **上游**: 設備供應商, 原廠 (電子零組件/消費品).
*   **下游**: 政府標案 (水處理/廢棄物), 一般通路.

## 主要客戶及供應商
### 主要客戶
*   **公部門**: 環保局, 水利局 (標案性質).
*   **一般市場**: 零售通路.

### 主要供應商
*   **設備商**: 環保設備製造商.
