# 4764 - 雙鍵

## 業務簡介
**板塊:** Basic Materials
**產業:** Specialty Chemicals
**市值:** 8,768 百萬台幣
**企業價值:** 9,361 百萬台幣

雙鍵化工 (Double Bond Chemical) 專注於 **塑膠添加劑** 與 **光固化材料**。
主要產品：
1.  **塑膠添加劑**: 紫外線吸收劑 (UVA)、光安定劑 (HALS)、抗氧化劑。應用於塑膠、塗料防老化。
2.  **UV 光固化材料**: 單體、寡聚體、光起始劑。應用於塗料、油墨、3D 列印。
3.  **數位印花油墨**: 紡織用數位噴印墨水。
4.  **電子材料**: **5G/6G 用銅箔基板 (CCL)** 添加劑、環氧樹脂硬化劑、LED 封裝膠。產品正轉型走向高階電子化學品。

## 供應鏈位置
*   **中游 (本體)**: **雙鍵 (4764)** - 化學添加劑/配方製造.
*   **上游**: 基礎化工原料 (苯酚, 醇類).
*   **下游**:
    *   **塑膠/塗料**: 台塑, 南亞 (塑膠加工), 塗料廠.
    *   **電子**: [[銅箔基板]] (CCL) 廠, LED 封裝廠.
    *   **紡織**: 數位印花染整廠.

## 主要客戶及供應商
### 主要客戶
*   全球塑膠與塗料大廠.
*   銅箔基板製造商.
