# 6870_騰雲

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 4,623 百萬台幣
**企業價值:** 5,000 百萬台幣

騰雲科技服務股份有限公司 (6870) 是一家專為大型零售商與購物中心提供「客流、商流、金流」全方位數位轉型解決方案的軟體科技(SaaS)公司。騰雲科技打造了強大的「All-in-One」雲端服務平台，能夠無縫整合百貨商場最頭痛的異質系統：從前端的各類行動支付(Apple Pay、Line Pay等)、POS收銀、到後端的電子發票、會員紅利(CRM)與櫃位帳務結算。更透過導入AI與大數據技術，為商場提供深度的客流分析與精準行銷。憑藉極高的系統穩定度，騰雲幾乎壟斷了台灣大型百貨與購物中心的資訊金流系統。

## 供應鏈位置
*   **上游**: 大型公有雲基礎設施平台服務商(如AWS、Azure、GCP)、AI大數據演算法技術夥伴及收銀POS硬體商。
*   **中游**: **騰雲科技** (O2O虛實融合零售雲端客商金流整合解決方案、AIoT智慧建築資訊及大數據分析SaaS服務)。
*   **下游**: 大型百貨連鎖集團、綜合型購物中心、連鎖餐飲/零售通路、大型量販超市及跨國飯店渡假村。

## 主要客戶及供應商
### 主要客戶
*   以台灣百貨三巨頭(新光三越、遠東百貨、SOGO)為核心，並跨足海外如日本三井與AEON等國際集團。

### 主要供應商
*   全球各大雲端伺服器供應原廠，並與日本Hitachi(日立)、VINX等大型系統資訊商結盟為海外代理。
