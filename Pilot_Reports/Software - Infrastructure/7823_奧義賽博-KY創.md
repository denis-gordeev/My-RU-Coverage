# 7823 - 奧義賽博-KY創

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 3,500 百萬台幣
**企業價值:** 2,791 百萬台幣

奧義智慧科技股份有限公司 (7823) 是台灣少見能以「純正AI資安血統」反攻國際舞台，幫企業與國防部隊架設大腦防護罩的黑帽剋星。傳統防毒軟體只能辨識已知的死板病毒碼，但面對隱蔽且狡猾的國家級APT駭客網軍，往往束手無策。奧義智慧自主研發出獨步純軟AI大腦，能在一分鐘內串聯高達百萬個端點軌跡，像超級警探般自動拼湊出駭客的攻擊意圖與入侵路徑。這套軟體不僅保護著台灣絕大多數的金控銀行、指標性晶圓半導體廠與政府軍工單位，現在更將防禦網延伸至新一代的大語音模型(LLM)與無人機干擾反制對抗。

## 供應鏈位置
*   **上游**: 提供龐大伺服器與AI演算法機器學習算力底層之國際頂級公有雲服務平台(如AWS, GCP)，以及各種開源AI模型/資安威脅情資資料庫供應商。
*   **中游**: **奧義智慧科技(CyCraft)** (純軟體AI資安服務原廠、專注於端點偵測回應(EDR)、外部攻擊面管理(EASM)、AI威脅鑑識及大型語言模型(LLM)專屬AI防火牆開發)。
*   **下游**: 遭受全球高階網路犯罪集團與國家級網軍(APT)頻繁攻擊與勒索之高價值標的，包含政府軍工單位、晶圓半導體廠、大型金控銀行與中大型企業集團。

## 主要客戶及供應商
### 主要客戶
*   累積逾三百家極具指標性的黃金客戶，包含國內外首屈一指的晶圓代工巨頭、高標準金融銀行業、國防部隊，更透過代理商積極插旗日本(如日立協作)與東南亞市場。

### 主要供應商
*   身為「純正軟體原廠」，無實體零件供應鏈，但其海內外之營業推廣高度依賴強有力的「資安代理經銷商」(如能率集團旗下宏羚等IT整合商)協助落地佈署。
