# 6878_歐付寶

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 977 百萬台幣
**企業價值:** 866 百萬台幣

歐付寶電子支付股份有限公司 (6878) 隸屬於綠界科技母集團，是台灣第一家取得金管會核發專營「電子支付」機構營業執照的先驅企業。歐付寶致力於為台灣店家建立安全且便捷的無現金交易環境，提供線上與線下的多元收款解決方案。透過串接信用卡、銀行帳戶及提供個人P2P轉帳服務，消除了微型商家在金流建置與收款上的痛點。近年，歐付寶更整合電子發票開立功能，並積極與國際金融機構、跨境支付聯盟及其他本土電子支付(如街口、一卡通)進行跨平台整合與流量共享，擴大無現金生態圈。

## 供應鏈位置
*   **上游**: 台灣各大銀行機構(帳戶綁定結算)、國際信用卡發卡組織(VISA/Mastercard等)、聯徵中心與相關資安認證體系。
*   **中游**: **歐付寶** (提供O2O線上/實體店面跨平台行動支付收款服務、個人電子錢包儲值轉帳及微型店家電子發票雲端服務)。
*   **下游**: 全台灣廣泛分布之中小企業/微型實體商家、電子商務網站營運商、外送平台及各年齡層之消費大眾。

## 主要客戶及供應商
### 主要客戶
*   全台數十萬家導入其收單條碼之中小型商戶、夜市攤商、計程車司機及一般註冊開戶之活躍消費者。

### 主要供應商
*   台灣在地的電信營運商、大型雲端伺服器機房及提供防偽詐騙與身分認證(KYC)的各類FinTech技術支援商。
