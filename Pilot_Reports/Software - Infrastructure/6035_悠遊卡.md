# 6035_悠遊卡

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 3,846 百萬台幣
**企業價值:** 2,851 百萬台幣

鼎豐光電科技有限公司 (6035) 是一家專注於再生能源與綠能系統開發的企業。主要業務涵蓋再生能源自用發電設備的建置(如太陽能光電)、熱能供應以及溫泉取供等綠色工程與維運服務。隨著全球淨零碳排趨勢，鼎豐光電致力於為企業及機構提供客製化的綠電解決方案與能源轉型基礎建設。

## 供應鏈位置
*   **上游**: 太陽能板、逆變器、儲能電池模組及相關綠能機電設備製造商。
*   **中游**: **鼎豐光電** (再生能源發電系統建置、熱能供應、綠電營運維護)。
*   **下游**: 尋求能源轉型之企業、政府機構、大型廠房及社區。

## 主要客戶及供應商
### 主要客戶
*   有建置屋頂型或地面型太陽能需求之企業戶、綠電採購方及熱能/溫泉供應需求之機構。

### 主要供應商
*   國內外各大光電模組廠、機電工程統包商(EPC)及儲能設備供應商。
