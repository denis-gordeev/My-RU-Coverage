# 7547 - 碩網

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 1,843 百萬台幣
**企業價值:** 924 百萬台幣

碩網資訊股份有限公司 (7547) 是台灣在「對話式AI」與「自然語言處理(NLP)」領域的軟體系統霸主。在AI浪潮席捲全球的時代，碩網專精於將生硬的AI大語言模型(如ChatGPT、Google Gemini)落地轉化為企業能實際運用的「智能大腦」。其核心產品如SmartRobot(智能客服系統)在台灣金融業擁有超過八成壓倒性的市佔率。碩網不僅幫企業打造能24小時精準應答的虛擬客服，更將AI導入企業內部(SmartWork、KMS)，協助員工快速爬梳海量內部文件與進行智能決策。它是推動台灣金融與公部門展開生成式AI數位轉型的關鍵後台大將。

## 供應鏈位置
*   **上游**: 提供底層龐大算力與開源/閉源大型語言模型(LLM)之全球頂尖雲端基礎設施原廠巨擘(如微軟Azure OpenAI、AWS、Google Cloud)及NVIDIA等運算平台。
*   **中游**: **碩網資訊(Intumit)** (企業級對話式AI軟體產品開發、自然語言處理NLP/機器學習演算法應用整合、生成式AI(GenAI)建置與顧問服務)。
*   **下游**: 亟需導入AI以降低人力成本並提升服務體驗的各大金控銀行、壽險業、證券商、高科技製造業(如台積電等內部Helpdesk)、以及各級政府機關。

## 主要客戶及供應商
### 主要客戶
*   以台灣大型金控集團(市佔逾八成)為絕對主力客戶群，並涵蓋台積電等高科技大廠及眾多公部門專案，且正積極擴張日本大型保險金融市場。

### 主要供應商
*   最核心且高度依賴的技術與雲端平台供應商即為[[微軟]](Microsoft)，碩網為其極重要之ISV(獨立軟體供應商)戰略合作夥伴。
