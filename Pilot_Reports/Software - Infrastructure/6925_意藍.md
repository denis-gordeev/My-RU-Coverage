# 6925_意藍

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 1,628 百萬台幣
**企業價值:** 1,264 百萬台幣

意藍資訊股份有限公司 (6925) 是台灣目前在「大數據語意分析」與「AI社群聲量監測」領域市佔第一的SaaS軟體霸主。在資訊爆炸且網軍/假訊息充斥的時代，意藍的核心產品「OpView社群口碑資料庫」能全天候自動爬梳全台各大論壇、新聞及社群媒體，利用自然語言處理(NLP)及大型語言模型(LLM)精準判讀網民情緒與關鍵字。這成為台灣各大消費品牌、大型金融集團甚至國安情報單位和地方政府進行輿情監測、危機公關與市場行銷不可或缺的「AI智庫」與「風向球」。

## 供應鏈位置
*   **上游**: 提供海量算力與雲端儲存空間的大型公有雲平台(如AWS、Google Cloud)、以及部分外部開源大語言模型(LLM)演算法。
*   **中游**: **意藍資訊** (AI語意分析平台OpView、社群大數據分析監測服務、企業智能商務搜尋引擎SaaS軟體開發與訂閱服務)。
*   **下游**: 高度依賴市場口碑與品牌形象的B2C跨國企業/零售業、大型金控集團、各級中央與地方政府行政機關及國安情報單位。

## 主要客戶及供應商
### 主要客戶
*   超過八成的雲端經常性訂閱收入來自台灣頂級金融科技業、電信巨擘以及公部門的「輿情防護與政策風向觀測」專案。

### 主要供應商
*   高度仰賴全球領先的雲端基礎設施服務原始供應商，為其極大吞吐量的網頁爬蟲與語意分析模型提供底層算力。
