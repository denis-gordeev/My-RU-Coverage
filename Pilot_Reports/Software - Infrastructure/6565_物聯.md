# 6565_物聯

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 457 百萬台幣
**企業價值:** 344 百萬台幣

物聯智慧股份有限公司 (6565，ThroughTek) 是一家專注於[[物聯網]]([[IoT]])與人工智慧聯網(AIoT)領域的雲端軟體平台服務商(PaaS/SaaS)。其核心競爭力在於自主研發的「Kalay」雲端平台，專精於海量高畫質影像資料的安全連線與跨裝置無縫整合。物聯智慧協助傳統硬體設備製造商或品牌商快速將產品(如安防攝影機、智慧家電、嬰兒監視器)連上雲端，提供影像串流、雲端錄影存儲及[[AI]]邊緣運算等加值服務。近年來積極將重心轉向北美DIY安防品牌及日本大型電信通路，以標準化軟體授權模式創造穩定的經常性收入。

## 供應鏈位置
*   **上游**: 雲端基礎設施供應商(如AWS/Azure等公有雲)、[[AI]]演算法授權商及網路安全技術提供商。
*   **中游**: **物聯智慧** (AIoT[[物聯網]]影像[[雲端服務]]平台Kalay之運營、軟體韌體開發服務與授權)。
*   **下游**: 全球[[消費性電子]]品牌商、DIY安控設備製造商、系統整合商(SI)、電信營運商及一般終端消費者。

## 主要客戶及供應商
### 主要客戶
*   北美主要的DIY家用安防品牌大廠，以及日本前三大電信商之一(提供其智慧家庭解決方案)。

### 主要供應商
*   亞馬遜AWS等全球大型公有雲平台基礎設施供應商(IaaS層級的支持)。
