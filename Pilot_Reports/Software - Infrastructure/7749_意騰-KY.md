# 7749 - 意騰-KY

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 19,640 百萬台幣
**企業價值:** 16,587 百萬台幣

意騰科技股份有限公司 (7749) 是台灣在邊緣運算層級將「機器聽覺」發揮到極致的AI聲學黑科技公司。在吵雜的廠房或行進的車廂中，如何讓語音助理或耳機只聽取你的聲音而忽略環境噪音？意騰專攻極低功耗的「邊緣AI神經網路聲學演算法」與專用ASIC晶片。他們不僅將極其複雜的語音降噪、聲紋辨識與回音消除技術壓縮到算力極為有限的小型終端設備(如TWS藍牙耳機、筆電或車載儀表)中，更成為NVIDIA與聯發科在AI PC晶片計畫中指定的「聲學大腦」供應商，是賦予百萬台智慧裝置擁有純淨「聽覺」的靈魂推手。

## 供應鏈位置
*   **上游**: 提供高解析度微型麥克風/音訊收音陣列元件感測廠、矽智財(IP)授權商以及晶圓代工與後段封測大廠(用於其ASIC晶片量產)。
*   **中游**: **意騰科技** (專注於AI邊緣運算之類神經網路演算法開發、高階語音辨識與聲學降噪處理軟體IP授權，以及低功耗AI聲學邊緣運算專用ASIC晶片設計)。
*   **下游**: 全球頂級之消費性電子品牌廠(如高階藍牙耳機、AI PC、智慧音箱)、車載導航與通訊娛樂系統Tier-1供應商，以及全球主要的手機通訊晶片主控原廠。

## 主要客戶及供應商
### 主要客戶
*   以全球一線消費性電子大廠與AI晶片龍頭為服務對象；並成功打入[[輝達]](Nvidia)與[[聯發科]](MediaTek)之AI PC新世代晶片聲學供應鏈，亦積極將觸角延伸至日本車用大型供應商(如DENSO)。

### 主要供應商
*   作為晶片設計與演算法授權商(Fabless)，其供應鏈命脈高度仰賴[[台積電]]等晶圓代工廠的產能支持，以及與關鍵通訊主晶片(如聯發科)的緊密戰略適配。
