# 7819 - 精誠金融

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 21,480 百萬台幣
**企業價值:** 2,128 百萬台幣

精誠金融科技股份有限公司 (7819) 是台灣線下實體店面「刷卡機與行動支付金流」的絕對統治者。由台灣軟體業霸主精誠集團(6214)分拆而出的它，掌握著極度驚人的線下通路資源——全台灣高達七成的刷卡終端設備銷售，以及一半以上的維修保養皆出自其手。不論你在便利商店買咖啡、量販店結帳或餐廳刷卡，背後那台連線極速、硬體堅固的刷卡機，以及其連通銀行的信用卡收單核心系統，幾乎都是精誠金融科技的傑作。它正挾帶龐大的實體據點優勢，瘋狂吞噬包含第三方支付與電子發票在內的OMO(全通路)零售金流大餅。

## 供應鏈位置
*   **上游**: 全球三大頂級刷卡終端硬體設備原廠(如美國VeriFone、中國百富PAX、台灣虹堡Castles)，以及提供金流後台加密加密運算模組之系統軟體商。
*   **中游**: **精誠金融科技** (代理銷售多國品牌刷卡機端末設備、全台刷卡機維運暨特店商化服務、信用卡收單系統建置以及電子發票與第三方支付等零售金流加值服務)。
*   **下游**: 全台灣數以萬計有實體信用卡或多元支付收款需求之零售門市、連鎖超商餐廳、大型量販店，以及負責佈建這些端末收款特約據點的發卡與收單銀行。

## 主要客戶及供應商
### 主要客戶
*   以各大「信用卡收單銀行」(如聯邦、新光等)及大型連鎖零售集團(如萊爾富)為極度黏著之傳統靠山；並逐步向中小型商家推展第三方行動支付解決方案。

### 主要供應商
*   硬體方面高度仰賴其總代理之國際一線刷卡機原廠(如VeriFone)；企業營運與後台系統骨幹則受到其最大控股母公司台灣軟體龍頭[[精誠資訊]]的全力撐腰。
