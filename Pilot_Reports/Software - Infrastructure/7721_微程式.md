# 7721 - 微程式

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 2,647 百萬台幣
**企業價值:** 1,956 百萬台幣

微程式資訊股份有限公司 (7721) 是台灣將「數位支付」與「物聯網神經脈絡」深植於你我日常生活的隱形幕後推手。當你拿著悠遊卡嗶聲騎走YouBike公共自行車，或在停車場自動扣款時，背後那套全天候穩定且精準感測的支付終端大腦，極高機率來自微程式。他們不賣消費品，而是以「技術研發為本」的輕資產架構，將其極強的RFID、電控與雲端物聯網實力，灌注在電動自行車(E-Bike)及半導體廠房自動化監控感測等硬核產業。它是推動台灣傳統硬體設備廠走向「智慧化」與「服務化」的最強軟硬整合魔法師。

## 供應鏈位置
*   **上游**: 各種高規無線通訊晶片(RFID/NFC/藍牙等)、微處理器(MCU)半導體原廠、以及提供基礎運算虛擬機與網路頻寬之大型公有雲服務商。
*   **中游**: **微程式資訊** (提供從公共交通數位電子支付終端、智慧E-Bike機電軟硬整合、到半導體產線智慧感測模組之跨領域AIoT客製化技術服務)。
*   **下游**: 全台各大型大眾交通運輸管理營運商、積極尋求升級高端智能電動機具之中小品牌自行車廠，以及致力提升無塵室良率之半導體精密製造廠。

## 主要客戶及供應商
### 主要客戶
*   以微笑單車(YouBike)及各大捷運公車票證系統為最知名實績；並獲得自行車製造霸主[[巨大]](Giant)與半導體載具龍頭[[家登]]的投資青睞與專案合作。

### 主要供應商
*   無獨佔性之重型製造廠房，其最核心之供應與護城河在於內部大量能寫代碼又懂機電整合(軟硬融合)的高階研發工程師腦力資源。
