# 6462_神盾

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 9,946 百萬台幣
**企業價值:** 14,339 百萬台幣

神盾股份有限公司 (6462，Egis Technology) 是一家從生體指紋辨識IC起家的台灣無晶圓廠(Fabless)[[半導體]]公司。過去神盾主要依賴韓國智慧型手機之指紋辨識感測器訂單。為擺脫單一產品風險，神盾近年發動多起併購與策略結盟(如併購乾瞻科技、芯鼎、安國等)，大舉轉型為[[AI]]與高效能運算(HPC)之矽智財(IP)與ASIC設計服務平台。目前集團不僅切入先進製程(3k/5nm)的UCIe(小晶片互連)與高速[[記憶體]]介面IP，更攜手Arm架構開發[[伺服器]]等級的[[AI]]晶片。

## 供應鏈位置
*   **上游**: 矽晶圓製造、[[半導體]]晶圓代工廠、光罩服務及EDA電子設計[[自動化]]軟體供應商。
*   **中游**: **神盾** (指紋/生體辨識IC、[[AI]]/HPC高速運算矽智財IP授權及ASIC客製化晶片設計服務)。
*   **下游**: 智慧型手機品牌商、[[雲端服務]]供應商([[CSP]])、[[半導體]]IC設計公司(Fabless)及車廠。

## 主要客戶及供應商
### 主要客戶
*   三星(傳統指紋IC最大客戶)、韓國起亞汽車，並已取得美國Tier1雲端大廠([[CSP]])之IP驗證。

### 主要供應商
*   [[台積電]](先進製程設計定案合作夥伴)、三星晶圓代工及Arm(架構技術授權夥伴)。
